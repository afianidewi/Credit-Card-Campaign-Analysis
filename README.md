# Credit-Card-Campaign-Analysis

## Project Overview
This project analyzes the effectiveness of a credit card campaign conducted by a bank. The analysis focuses on evaluating whether the campaign significantly increased customer credit card usage and whether there are usage differences between male and female customers.

This project demonstrates skills in:
- Exploratory Data Analysis (EDA)
- Statistical Hypothesis Testing
- Business Insight Interpretation
- Data Visualization

> Dataset provided as part of a bootcamp assignment (not publicly shareable).

---

## Objectives

1. Perform Exploratory Data Analysis (EDA)
2. Evaluate campaign effectiveness:
   - Before vs 1 Month After Campaign
   - Before vs 2 Months After Campaign
3. Compare post-campaign usage:
   - Male vs Female (1 month)
   - Male vs Female (2 months)

---

## Tools & Libraries

- Python
- Pandas

---

## Statistical Methods Used

- Descriptive statistics
- Histogram visualization
- Paired Sample T-Test (`ttest_rel`)
- Independent Sample T-Test (`ttest_ind`)

---

## Business Interpretation

If p-value < 0.05:
- The campaign has statistically significant impact.

If p-value > 0.05:
- No significant evidence that campaign changes behavior.
